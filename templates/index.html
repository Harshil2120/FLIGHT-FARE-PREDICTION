Flight Fare Predicton Application

This is a Final Year project developed by Harshil Donga, Ashutosh Trivedi, Sonam Mahajan & Viraj Mahajan under the guidance of Dr.V.V.Kimbahune.It is based on machine learning domain where a predictive model is developed using supervised machine learning algorithms to give predicted value of flight price.

   This is Full End-to-End implemenation of machine learning model in the back-end which interacts with front-end to get user input data.The source code for this project can be accessed from below link.
